While retrieving the carbon platform, angular-carbonldp returned an HTTP.Errors.UnknownError.

This usually means that the server response didn't include the necessary CORS headers.

Double check that you have allowed all origins or added this app's domain to the list of allowed domains inside the instance of the platform.

Other common reasons are:

  • The configured Carbon platform host is wrong (right now it's set to {{ carbon.baseURI }})
  • The platform server doesn't support https and you didn't turn it off (use carbon.setSetting( 'http.ssl', false );)
  • The platform server is down or not accessible on {{carbon.resolve( ".system/platform/" )}} Click here to test it.
